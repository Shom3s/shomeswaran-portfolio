# Shomeswaran -- Software Developer Portfolio

This version keeps the layout and animation direction close to the uploaded sample video:

- dark hero with huge gradient heading
- centered magnetic 3D portrait
- scroll-based two-row project marquee
- dark About section with 3D decorative objects
- white Services/Price section
- sticky stacked Project cards
- testimonial cards
- white Contact section with black footer

Only the content was changed from 3D creator to software developer.

## Run on Windows

```cmd
npm config set registry https://registry.npmjs.org/
npm install --registry=https://registry.npmjs.org/
npm run dev
```

Open the local URL shown by Vite.
